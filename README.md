# Aplikasi Pengajuan Usulan Pelatihan dan LPJ

## Deskripsi Proyek

Aplikasi Pengajuan Usulan Pelatihan dan LPJ adalah sebuah sistem yang dirancang untuk memfasilitasi pengelolaan proses pengajuan usulan pelatihan dan Laporan Pertanggung Jawaban (LPJ) dalam sebuah perusahaan. Aplikasi ini membantu karyawan dan manajer dalam mengajukan, meninjau, dan menyetujui usulan pelatihan serta mempermudah proses pelaporan LPJ setelah pelatihan selesai.

## Fitur Utama

1. **Pengajuan Usulan Pelatihan**:
   - Formulir pengajuan usulan pelatihan yang mudah diisi oleh karyawan.
   - Sistem notifikasi otomatis untuk pemberitahuan status usulan.
   - Riwayat pengajuan usulan pelatihan yang dapat diakses kapan saja.

2. **Persetujuan Manajer**:
   - Dashboard manajer untuk meninjau dan menyetujui atau menolak usulan pelatihan.
   - Komentar dan feedback manajer langsung pada usulan yang diajukan.

3. **Pengajuan LPJ**:
   - Formulir LPJ yang terstruktur dengan baik untuk pelaporan pasca pelatihan.
   - Upload bukti dan dokumen pendukung LPJ.
   - Notifikasi pengingat untuk pengajuan LPJ tepat waktu.

4. **Manajemen Pengguna**:
   - Hak akses yang berbeda untuk karyawan dan manajer.
   - Pengelolaan data pengguna yang efisien.

## Teknologi yang Digunakan

- **Frontend**:
  - HTML, CSS, JavaScript untuk antarmuka pengguna yang responsif.

- **Backend**:
  - Bahasa pemrograman PHP untuk logika aplikasi.


- **Database**:
  - Sistem manajemen basis data menggunakan MySQL untuk penyimpanan data yang aman dan terstruktur.
